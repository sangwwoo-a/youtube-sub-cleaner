Youtube Sub Cleaner

유튜브 구독 채널이 너무 많아졌나요? 오랫동안 업로드가 없는 채널을 한눈에 찾아 빠르게 정리하세요.

Google 로그인 필요 광고 포함 무료
자세히 보기 →

주요 기능

휴면 채널 자동 감지 마지막 업로드 기준으로 오랫동안 활동이 없는 채널을 자동 분류
일괄 구독 취소 채널마다 들어갈 필요 없이 여러 채널을 한 번에 선택해 구독 취소
Google 공식 API 사용 YouTube Data API v3 사용. 비밀번호를 앱에 입력하지 않아요

주요 기능

날씨 연동 알람 비·눈 예보 시 알람을 자동으로 앞당겨 여유로운 아침을 만들어줌
맞춤 선행 시간 날씨 종류별로 몇 분 앞당길지 직접 설정 가능
완전 오프라인 동작 날씨 조회 외 모든 데이터는 기기 내부에만 저장. 계정 불필요

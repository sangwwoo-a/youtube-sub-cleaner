주요 기능

스마트한 기상을 위한 모든 기능을 담았어요.

🌦️

날씨 연동 알람

비 또는 눈이 예보된 날, 알람을 자동으로 앞당겨 여유로운 아침을 만들어줘요.

⏰

맞춤 선행 시간

비 올 때 몇 분, 눈 올 때 몇 분 앞당길지 날씨 종류별로 직접 설정할 수 있어요.

📍

위치 기반 날씨

현재 위치 또는 직접 지정한 위치의 날씨를 기준으로 알람을 조정해요.

🔁

반복 알람

요일별 반복 설정을 지원해요. 평일 출근 알람을 한 번만 등록하면 매주 자동으로 작동해요.

🔔

이동 알림

알람이 앞당겨졌을 때 알림으로 바로 알려줘요. 몇 분 이동됐는지 이유와 함께 표시돼요.

🔒

로그인 불필요

Google이나 별도 계정 로그인 없이 바로 사용할 수 있어요. 모든 데이터는 기기 안에만 저장돼요.

안전하게 사용하세요

개인정보와 기기 보안을 최우선으로 생각해요.

✓로그인이 필요 없어요. 계정 가입이나 개인 인증 없이 바로 사용할 수 있습니다.
✓외부 서버에 데이터를 저장하지 않아요. 알람 설정은 사용자 기기 안에만 저장됩니다.
✓위치 정보는 날씨 조회에만 사용해요. 위치 데이터는 OpenWeatherMap API 호출 후 기기에 보관되지 않습니다.
✓앱 삭제 시 모든 데이터가 즉시 삭제돼요. 별도의 계정 탈퇴 절차가 필요 없습니다.

오늘 밤부터 스마트하게 일어나세요

무료로 사용할 수 있어요. 로그인도 필요 없습니다.

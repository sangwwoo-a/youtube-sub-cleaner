이런 경험 있으신가요?

구독 채널이 너무 많아 관리가 어렵고, 오래된 채널들이 피드를 가득 채우고 있지 않으신가요?

😴

오래된 채널

1년 이상 업로드가 없는 채널들이 구독 목록을 가득 채우고 있어요

😩

하나씩 취소

채널마다 하나씩 구독 취소하는 건 너무 귀찮고 시간이 오래 걸려요

📺

지저분한 피드

관심 없는 채널들이 진짜 보고 싶은 콘텐츠를 가리고 있어요

사용 방법

4단계로 구독 목록을 깔끔하게 정리하세요.

1

구글 로그인

YouTube 계정에 안전하게 로그인하세요

→
2

필터 설정

몇 개월 이상 비활성화된 채널을 필터링할지 선택하세요

→
3

자동 분석

앱이 각 채널의 마지막 업로드 날짜를 자동으로 확인해요

→
4

일괄 취소

선택한 채널들을 한 번에 구독 취소하세요

주요 기능

구독 관리에 필요한 모든 것.

🔍

실시간 채널 분석

구독 목록을 가져와 각 채널의 마지막 업로드 날짜를 실시간으로 분석해요.

🎨

한눈에 색상 구분

삭제할 채널은 빨간색, 유지할 채널은 초록색으로 표시되어 한눈에 파악할 수 있어요.

☑️

전체 선택 / 해제

자동 선택된 목록에서 시작하거나, 구독 취소할 채널을 직접 골라보세요.

📅

기간 필터

비활성화 기준을 1개월부터 24개월까지 자유롭게 설정하세요.

🎯

3가지 정리 모드

오래된 업로드 순, 구독 날짜 순, 또는 직접 선택 — 필요에 맞게 골라보세요.

⚡

일괄 구독 취소

선택한 채널들을 한 번에 구독 취소해요. 짧은 광고 후 실행됩니다.

↩️

구독 복구하기

실수로 취소한 채널도 걱정 없어요. 이 앱에서 취소한 채널을 언제든지 다시 복구할 수 있어요.
구독 해지 이력은 암호화되어 기기에 영구 보관되고, 앱 삭제 시 함께 삭제됩니다.

개인정보 보호

회원님의 개인정보와 계정 보안을 소중히 여깁니다.

✓비밀번호 불필요. 구글 공식 OAuth 2.0만 사용합니다.
✓외부 서버에 데이터를 저장하지 않습니다. 모든 데이터는 기기에만 저장됩니다.
✓선택한 채널만 구독 취소됩니다. 자동 삭제는 없습니다.
✓Google 계정 설정에서 언제든지 액세스 권한을 취소할 수 있습니다.
